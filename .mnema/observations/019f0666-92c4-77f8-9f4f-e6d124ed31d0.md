(2) logger.ts loggerConfig(env) — Pino do Fastify: dev→pino-pretty/debug, prod→JSON/info; usado em buildApp quando logger:true. (3) telemetry.ts — NodeSDK OTel, startTelemetry() ANTES do Fastify (no server.ts main, antes do buildApp), exporter por OTEL_EXPORTER (console/otlp-grpc/none); withSpan(name,fn) para os spans customizados (nf.parse/nf.validate/nf.graph.merge/job.process/export.generate). Instrumentações automáticas fastify/http/neo4j/ioredis ficam para quando integrar os pacotes de instrumentation (não instalei todos para não inchar deps; o SDK base + spans manuais cobrem o AC). 7 testes unit.
