Baseline de bundle do dashboard (vite build) para a auditoria da NOTA-95: PRÉ-redesign (T1) — JS 1.029,63 kB (gzip 312,34 kB), CSS 20,42 kB (gzip 3,90 kB). PÓS-fundação (T2/NOTA-85, Tailwind v4 + 24 componentes shadcn ainda não importados) — JS 1.029,63 kB (gzip 312,34 kB, inalterado: ui/* tree-shaken), CSS 84,11 kB (gzip 14,69 kB: preflight + utilities detectadas nos ui/* + tw-animate + CSS legado), fontes Inter Variable ~218 kB em subsets woff2 (latin 48,25 kB é o que carrega de fato). Nota de setup: @tailwindcss/vite@4.3.2 instalou limpo com Vite 8.1 (sem conflito de peer, sem precisar aprovar build script do oxide); o que o pnpm 10 ignorou foi esbuild@0.28.1, sem efeito no build (rolldown). O shadcn add exigiu utils.ts manual (o init foi pulado — components.json autorado à mão) e clsx+tailwind-merge explícitos; o CLI anexou um bloco .dark de sidebar ao globals.css que foi removido (ADR-7: dark é [data-theme='escuro']).
