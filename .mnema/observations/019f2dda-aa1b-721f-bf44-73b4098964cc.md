Gotcha novo (Fase 4): a aba 'Rede completa' (RedeGraph/Reagraph WebGL) dá 'SyntaxError: Unexpected token (' + ErrorBoundary APENAS no Vite dev server (localhost:5173), mas renderiza PERFEITO no build de produção (nginx, localhost:8080). É um problema de pré-bundling do three.js/Reagraph no esbuild-dev do Vite, não um bug de código. LIÇÃO: para verificar visualmente o grafo WebGL, screenshotar contra 8080 (após rebuildar o container), NUNCA contra 5173 — no dev a rede WebGL parece quebrada mas não está. As demais telas (Home/Explorer/tabelas) podem ser vistas no dev 5173 normalmente (HMR); só a rede WebGL exige o build. Casa com [[notagrafo-webgl-oklch-e-dist-stale]] e a memória docker/screenshot.
