NOTA-7: libxmljs2@0.37 é binário nativo e NÃO tem prebuild para Node 24 via instalação padrão do pnpm (build scripts bloqueados por segurança). Solução: adicionar 'libxmljs2' em pnpm.onlyBuiltDependencies no package.json e rodar pnpm install — aí o script install (prebuild-install || node-gyp rebuild) roda e o binário carrega. doc.validate(xsd) do libxmljs2 resolve os xs:include do XSD automaticamente quando o XSD é parseado com { baseUrl: caminhoDoXsd }. Fixture válida: gerar uma NFe v4.00 que passe no XSD oficial exige o bloco <Signature> (xmldsig) obrigatório — montei um Signature estruturalmente conforme (SignedInfo c14n/rsa-sha1/sha1 + Reference + SignatureValue + KeyInfo/X509Certificate) com valores base64 placeholder; o XSD valida só a ESTRUTURA da assinatura, não a criptografia. A NFe inteira (ide/emit/dest/det ICMSSN102/total/transp/pag) validou; só faltava a assinatura.
